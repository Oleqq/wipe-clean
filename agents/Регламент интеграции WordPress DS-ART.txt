1. Введение
Этот регламент описывает стандарты интеграции HTML/CSS/JS вёрстки с WordPress для сайтов услуг (без WooCommerce). Цель — обеспечить единообразие, качество и поддерживаемость кода.
1.1. Применимость
* Корпоративные сайты
* Сайты-визитки
* Лендинги
* Многостраничные сайты услуг
* Портфолио и блоги
1.2. Базовые требования
* Тема создаётся с нуля на базе Starter-темы Underscores (_s)
* ACF Pro — обязательный плагин для кастомных полей
* Код соответствует WordPress Coding Standards
* БЭМ-методология для CSS-классов
________________

3. Подключение стилей и скриптов
Все стили и скрипты подключаются через wp_enqueue_* в functions.php.
3.1. Базовое подключение
function theme_scripts() {
    // Глобальные стили (на всех страницах)
    wp_enqueue_style('theme-global', get_template_directory_uri() . '/css/style-global.css');
    wp_enqueue_style('theme-header', get_template_directory_uri() . '/css/header.css');
    wp_enqueue_style('theme-footer', get_template_directory_uri() . '/css/footer.css');


    // Библиотеки (если используются)
    wp_enqueue_style('swiper-css', get_template_directory_uri() . '/css/swiper.css');
    wp_enqueue_style('fancybox-css', get_template_directory_uri() . '/css/fancybox.css');


    // Скрипты
    wp_enqueue_script('swiper-js', get_template_directory_uri() . '/js/swiper.js', [], null, true);
    wp_enqueue_script('theme-script', get_template_directory_uri() . '/js/script.js', [], null, true);
}
add_action('wp_enqueue_scripts', 'theme_scripts');
3.2. Условная загрузка по шаблонам
Для оптимизации загружаем стили только там, где они нужны:
function theme_conditional_styles() {
    // Главная страница
    if (is_front_page()) {
        wp_enqueue_style('theme-index', get_template_directory_uri() . '/css/style.css');
    }


    // По шаблону страницы
    if (is_page_template('page-contacts.php')) {
        wp_enqueue_style('theme-contacts', get_template_directory_uri() . '/css/contacts.css');
    }


    if (is_page_template('page-about.php')) {
        wp_enqueue_style('theme-about', get_template_directory_uri() . '/css/about.css');
    }


    // Для CPT
    if (is_singular('blogs')) {
        wp_enqueue_style('theme-single-blog', get_template_directory_uri() . '/css/single.css');
    }


    // Страница 404
    if (is_404()) {
        wp_enqueue_style('theme-404', get_template_directory_uri() . '/css/page-404.css');
    }
}
add_action('wp_enqueue_scripts', 'theme_conditional_styles');
3.3. Передача данных в JavaScript (wp_localize_script)
Для передачи PHP-данных в JS используем wp_localize_script:
function theme_localize_scripts() {
    wp_localize_script('theme-script', 'theme_ajax', [
        'ajax_url' => admin_url('admin-ajax.php'),
        'nonce'    => wp_create_nonce('theme_nonce'),
        'home_url' => home_url(),
    ]);
}
add_action('wp_enqueue_scripts', 'theme_localize_scripts');
В JavaScript данные доступны через глобальный объект:
// В JS файле
console.log(theme_ajax.ajax_url); // /wp-admin/admin-ajax.php
console.log(theme_ajax.nonce);    // abc123...

4. Advanced Custom Fields (ACF)
ACF Pro — основной инструмент для создания кастомных полей. Все редактируемые данные сайта хранятся в ACF-полях.
4.1. ACF Options (Глобальные настройки)
Данные, общие для всего сайта (контакты, соцсети, логотипы), выносим в Options Page:
// Регистрация страницы настроек
if (function_exists('acf_add_options_page')) {
    acf_add_options_page([
        'page_title' => 'Настройки сайта',
        'menu_title' => 'Настройки сайта',
        'menu_slug'  => 'theme-settings',
        'capability' => 'edit_posts',
        'redirect'   => false,
        'icon_url'   => 'dashicons-admin-generic',
        'position'   => 2,
    ]);
}
Типичные поля для Options:
* Логотип (header и footer варианты)
* Контакты: телефон, email, адрес
* Ссылки на соцсети
* Ссылки на политику конфиденциальности
* Копирайт
* Настройки поп-апов (заголовки, описания)
Получение данных из Options:
// Используем второй параметр 'option'
$logo = get_field('logotip', 'option');
$phone = get_field('nomer_telefona', 'option');
$email = get_field('email', 'option');


// Очистка телефона от символов для href="tel:"
$phone_clean = preg_replace('![^0-9]+!', '', $phone);


// Использование в шаблоне
<a href="tel:+<?php echo $phone_clean; ?>"><?php echo $phone; ?></a>
4.2. Flexible Content (Секции страниц)
Для главной и других страниц с множеством секций используем Flexible Content:
// index.php или page-home.php
<?php
get_header();


if (have_rows('page_content')): // Имя поля Flexible Content
    while (have_rows('page_content')): the_row();
        
        // Секция Hero
        if (get_row_layout() == 'hero_section'):
            get_template_part('template-parts/section/hero');
        
        // Секция Услуги
        elseif (get_row_layout() == 'services_section'):
            get_template_part('template-parts/section/services');
        
        // Секция FAQ
        elseif (get_row_layout() == 'faq_section'):
            get_template_part('template-parts/section/faq');
        
        endif;
    endwhile;
endif;


get_footer();

4.4. Оптимизация ACF-запросов
Внутри циклов кешируем повторяющиеся вызовы get_field:
// ❌ Плохо — вызов на каждой итерации
while (have_rows('items')): the_row();
    echo get_field('global_setting', 'option'); // Каждый раз запрос
endwhile;


// ✅ Хорошо — кешируем перед циклом
$global_setting = get_field('global_setting', 'option');
while (have_rows('items')): the_row();
    echo $global_setting;
endwhile;

8. Меню навигации
Меню строим через ACF Repeater вместо стандартного wp_nav_menu(). Это даёт полный контроль над разметкой и позволяет добавлять кастомные поля.
8.1. Структура ACF-полей для меню
В ACF Options создаём Repeater с полями:
* nazvanie (Text) — текст ссылки
* ssylka (URL или Page Link) — ссылка
* est_podmenyu (True/False) — есть ли подменю
* podmenyu (Repeater) — вложенные пункты
8.2. Вывод меню
<?php if (have_rows('glavnoe_menyu', 'option')): ?>
<nav class="header__nav">
    <ul class="header__menu">
        <?php while (have_rows('glavnoe_menyu', 'option')): the_row();
            $name = get_sub_field('nazvanie');
            $link = get_sub_field('ssylka');
            $has_submenu = get_sub_field('est_podmenyu');
        ?>
        <li class="header__menu-item<?php echo $has_submenu ? ' has-dropdown' : ''; ?>">
            <a href="<?php echo esc_url($link); ?>"><?php echo esc_html($name); ?></a>
            
            <?php if ($has_submenu && have_rows('podmenyu')): ?>

9. AJAX-функционал
Все AJAX-запросы проходят через admin-ajax.php с обязательной проверкой nonce.
9.1. Регистрация AJAX-обработчика
// PHP: functions.php


// Для авторизованных и неавторизованных пользователей
add_action('wp_ajax_filter_posts', 'ajax_filter_posts');
add_action('wp_ajax_nopriv_filter_posts', 'ajax_filter_posts');


function ajax_filter_posts() {
    // 1. Проверка nonce — ОБЯЗАТЕЛЬНО
    if (!check_ajax_referer('theme_nonce', 'nonce', false)) {
        wp_send_json_error(['message' => 'Ошибка безопасности']);
    }
    
    // 2. Получение и санитизация данных
    $category = isset($_POST['category']) ? sanitize_text_field($_POST['category']) : '';
    $page = isset($_POST['page']) ? absint($_POST['page']) : 1;
    
    // 3. Формируем запрос
    $args = [
        'post_type'      => 'blogs',
        'posts_per_page' => 9,
        'paged'          => $page,
        'post_status'    => 'publish',
    ];
    
    if ($category && $category !== 'all') {
        $args['tax_query'] = [[
            'taxonomy' => 'blog_category',
            'field'    => 'term_id',
            'terms'    => intval($category),
        ]];
    }
    
    $query = new WP_Query($args);
    
    // 4. Рендерим HTML
    ob_start();
    if ($query->have_posts()):
        while ($query->have_posts()): $query->the_post();
            get_template_part('template-parts/blog', 'card');
        endwhile;
        wp_reset_postdata();
    else:
        echo '<p class="no-posts">Записей не найдено</p>';
    endif;
    $html = ob_get_clean();
    
    // 5. Отправляем ответ
    wp_send_json_success([
        'html'      => $html,
        'max_pages' => $query->max_num_pages,
        'found'     => $query->found_posts,
    ]);
}
9.2. JavaScript AJAX-запрос
// JS: script.js


document.addEventListener('DOMContentLoaded', () => {
    const filterBtns = document.querySelectorAll('.filter__btn');
    const postsContainer = document.querySelector('.posts__grid');
    
    filterBtns.forEach(btn => {
        btn.addEventListener('click', function() {
            const category = this.dataset.category;
            
            fetch(theme_ajax.ajax_url, {
                method: 'POST',
                headers: {
                    'Content-Type': 'application/x-www-form-urlencoded',
                },
                body: new URLSearchParams({
                    action: 'filter_posts',
                    nonce: theme_ajax.nonce,
                    category: category,
                    page: 1
                })
            })
            .then(response => response.json())
            .then(data => {
                if (data.success) {
                    postsContainer.innerHTML = data.data.html;
                } else {
                    console.error(data.data.message);
                }
            })
            .catch(error => console.error('Error:', error));
        });
    });
});

add_action('wp_ajax_load_more', 'ajax_load_more');

function ajax_load_more() {

    $args = [
        'post_type'      => 'blogs',
        'posts_per_page' => 9,
        'paged'          => $page,
    ];
    
    if ($category !== 'all') {
        $args['tax_query'] = [[
            'taxonomy' => 'blog_category',
            'field'    => 'term_id',
            'terms'    => intval($category),
        ]];
    }
    
    $query = new WP_Query($args);
    
    ob_start();
    while ($query->have_posts()): $query->the_post();
        get_template_part('template-parts/blog', 'card');
    endwhile;
    wp_reset_postdata();
    $html = ob_get_clean();
    
    wp_send_json_success([
        'html'      => $html,
        'max_pages' => $query->max_num_pages,
    ]);
}
⚠️ Всегда используйте check_ajax_referer() или wp_verify_nonce() для проверки безопасности AJAX-запросов.

10.2. Кастомная разметка формы
CF7 оборачивает элементы в теги <p> и <span>. Для полного контроля над разметкой можно их удалить:
// Удаление <p> и <span> из Contact Form 7
add_filter('wpcf7_form_elements', function($content) {
    $content = preg_replace('/<span[^>]*>(.*?)<\/span>/', '$1', $content);
    $content = preg_replace('/<p\b[^>]*>/', '', $content);
    $content = str_replace('</p>', '', $content);
    return $content;
});
